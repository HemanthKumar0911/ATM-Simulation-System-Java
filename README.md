# ATM-Simulation-System-Java
A simple ATM simulation project developed in Java using Object-Oriented Programming concepts. This project allows users to securely log in using a PIN and perform basic banking operations such as withdrawing money, depositing money, and checking account balance. It is designed to demonstrate core Java and OOP fundamentals in an easy and practical .

# ATM Simulation Project with PIN Verification

## Description
This is an Object-Oriented ATM Simulation Project developed in Java.
The project includes secure PIN verification before allowing users
to perform banking operations.

## Features
- PIN authentication
- Withdraw money
- Deposit money
- Check balance
- Exit system

## OOP Concepts Used
- Encapsulation
- Constructor
- Methods
- Object Interaction

## How to Run
javac ATM.java

java ATM
